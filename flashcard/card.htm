Function Notation
info_outline

Function notation is a method of writing algebraic variables as functions of other variables.

What will a standard flashcard look like? See above

How will the user interact with it? The user will click the image to open the visual or the i icon to see the definition.

How will you get from one card to the next? The cards are all on the same page, so you would just scroll down the page and interact with a flashcard.

Will there be a list of topics? Yes, there will be a list of topics on the right.

Will it be searchable? How do we do that? Yes- there will be a search bar where you can type in a term and it will look for a flashcard.

What about related topics? At the moment we dont believe we will implement this feature.

An i icon with more information, hidden at first so as not to distract from the main? Yes, this will be used to reveal the definition once the visual is clicked or you can simply click the i to get both the definition and the visual.

A share icon to easily copy the link to this particular card? We will put the share icon next to the i icon as far as we know of.
